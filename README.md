
## Установка

Клонируйте репозиторий и перейдите в папку проекта:

git clone <URL-репозитория>

Создайте и активируйте виртуальное окружение:

python -m venv venv
.\venv\Scripts\activate

Установите зависимости:

pip install ultralytics opencv-python numpy shapely

## Структура проекта

config/

  rtsp_analyzer.py         # Анализ видео без масок

## Модель

Система использует модель YOLO11m-seg для детекции с масками. Модель скачивается автоматически при первом запуске.

Скачать модель вручную:

python -c "from ultralytics import YOLO; YOLO('yolo11m-seg.pt')"

## Использование

### 1. Разметка парковочных мест

Перед анализом необходимо разметить парковочные места на изображении:

python utils/marker.py

Инструкция по разметке:
- Левый клик мыши - добавить точку угла места
- Правый клик мыши - завершить ввод места (нужно минимум 4 точки)
- Q - сохранить и выйти

### 2. Анализ изображения

После разметки можно проанализировать изображение:

python src/main.py

Результат будет сохранён в data/result_yolo11m_pixelwise.jpg

### 3. Тестирование модели

Для проверки работы модели с масками:

python test_yolo.py

## Настройка параметров

В файле src/main.py можно настроить следующие параметры:

overlap_threshold = 0.3        # Порог перекрытия маски (0.2-0.5)
model_name = 'yolo11m-seg.pt'  # Модель для детекции
imgsz = 640                   # Размер входного изображения
